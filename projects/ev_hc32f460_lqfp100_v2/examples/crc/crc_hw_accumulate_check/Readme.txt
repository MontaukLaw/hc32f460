平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程实现了CRC16和CRC32的累加校验功能。

说明：
1）硬件CRC按帧写入指定的数据；
2）硬件CRC写入校验码
3）根据CRC结果状态，判断CRC累加校验结果是否正确

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程，编译、下载并运行程序；
2）观察LED蓝灯常亮，表示硬件CRC累加校验结果正确；

================================================================================
注意
================================================================================
LED状态说明：
   a）蓝色LED灯亮，表示硬件CRC累加校验结果正确。
   b）红色LED灯亮，表示硬件CRC累加校验结果错误。
