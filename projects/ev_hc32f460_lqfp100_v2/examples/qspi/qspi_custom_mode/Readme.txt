平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示QSPI读写板载Flash功能。

说明：
本样例设置QSPI为自定义标准读模式，主要介绍采用直通模式来实现多线的读写功能。每按一次K2
则触发一次擦除flash、写入数据到flash、从flash读取数据，再将读写的数据进行比较，
相同的LED_BLUE亮，不同则LED_RED亮。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F460_LQ100_Rev2.0

辅助工具:
---------------------
Micro USB数据线

================================================================================
使用步骤
================================================================================
1）使用USB线连接电脑及目标板的J8 DAP端口；
2）打开工程并重新编译，启动IDE的下载和调试功能；
3）按下按键K2，触发擦除、读写对比功能，读写内容相同则LED_BLUE亮，不同则LED_RED
亮，可重复按下按键K2多次测试读写功能。

================================================================================
注意
================================================================================
本样例中QSSN引脚需设置为普通IO模式，通过软件控制其输出高低电平来实现功能。
